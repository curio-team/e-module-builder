Aan de slag

Content toevoegen

Vul de map content/ met jouw module-inhoud en bouw de site opnieuw.

  1. 1

    Module-metadata instellen

    Bewerk content/module.md — stel naam, aantal weken, taal en oefenmodus in.

  2. 2

    Weekcontent schrijven

    Maak per week een map content/week1/ enz. met theory.md, quiz.md, exercises.md en homework.md.

  3. 3

    Meetmomenten toevoegen (optioneel)

    Plaats content/assessments/theory-assessment.md en practical-assessment.md met vragen in YAML frontmatter.

  4. 4

    Bouwen en bekijken

    npm run dev      # ontwikkelserver
npm run build    # productie-build
npm run preview  # preview van dist/
